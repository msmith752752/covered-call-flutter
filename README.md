# 📈 Yield Pilot

**Navigate Covered Call Income With Clarity**

Yield Pilot is a Flutter-based mobile application backed by a Python FastAPI service that analyzes real market data and generates covered call strategies with clear risk and return insights.

---

## 🚀 Features

- 📊 Covered call strategy recommendations  
- 💰 Estimated premium income  
- 📉 Break-even price calculation  
- 📈 Max profit estimation  
- ⚖️ Risk labeling (Conservative, Balanced, Aggressive)  
- 💾 Input persistence (symbol + shares saved locally)  
- 📱 Clean, mobile-first UI  

---

## 🧠 How It Works

1. Enter a stock symbol (e.g. AAPL)  
2. Enter number of shares owned  
3. The app:
   - Fetches live market data  
   - Filters option chains  
   - Returns 3 ranked covered call strategies  

---
